## 📌 Project Description
This is a web application built with **Flask** that presents 3 famous Korean thriller movies.

Users can:
- Register an account
- Login securely
- View movies
- Add new movies
- Edit movies
- Delete movies

⚠️ Only logged-in users can perform CRUD operations.

---

## 🎥 Movies Included
- Oldboy (2003)
- The Gangster, the Cop, the Devil (2019)
- The Call (2020)

---

## ⚙️ Features
✔ User Registration & Login (Flask-Login)  
✔ Password-protected system  
✔ CRUD operations (Create, Read, Update, Delete)  
✔ Image support for movies  
✔ Responsive UI (Bootstrap 5)  
✔ About page
