About Website

Korean Thriller Movies

This website is dedicated to three popular Korean thriller movies: Oldboy, The Gangster, The Cop, The Devil and The Call.

Users can register, log in and manage movie information. Authorized users can add, edit and delete movies using CRUD operations.

The project was created using Flask, Flask-SQLAlchemy, Flask-Login, Flask-WTF, Bootstrap, HTML and CSS.
